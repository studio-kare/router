🚀 Quick Start

Use the public API key in your requests:

curl -X POST https://router.studiokare.nl/v1/chat/completions \
  -H "Authorization: Bearer sk_public_..." \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'

Or in Python:

from openai import OpenAI

client = OpenAI(
    api_key="sk_public_...",
    base_url="https://router.studiokare.nl"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)

📏 Rate Limits & Fair Use

  • 100 requests/second per IP (fairness limit)
  • 500 requests/second hard ceiling (5min cooldown if exceeded)
  • Shared daily pool of tokens (resets at midnight UTC)
  • Remaining pool shown in response headers: X-RateLimit-PoolRemaining
⚠️ Be Respectful
This is a shared resource powered by accumulated credits.
  • Don't abuse or load-test the service
  • No crawling or large-scale scraping
  • IPs that violate limits may be permanently banned
  • No SLA: service may go down

📚 Available Models

The router intelligently selects from:

  • OpenAI: gpt-4o, gpt-4-turbo, gpt-3.5-turbo
  • Anthropic: claude-3-5-sonnet, claude-3-opus
  • OpenRouter: llama-3.1, mistral, and many more

Selection is based on the privacy parameter (0-1) and model availability.

🔐 Privacy & Data

  • Your IP address is logged for rate limiting and abuse prevention
  • Request metadata is tracked (model, tokens, timestamp)
  • Logs are retained for 30 days
  • No authentication beyond API key needed

❓ FAQ

Is this really free?

Yes! It's powered by accumulated credits and offered as a gift to the community.

What happens when the pool runs out?

We'll either refill it or upgrade to a paid tier. You'll see a clear message if the pool is depleted.

Can I run my own instance?

Yes! Farmer is open source. Deploy it yourself, add your own API keys, and share with friends.

Why did my IP get banned?

IPs are banned for abuse (exceeding rate limits, large-scale scraping, DDoS attempts). If you believe it's a mistake, contact support.
